# flex-box model을 배워보자

[css-trics flex설명](https://css-tricks.com/snippets/css/a-guide-to-flexbox)

## container(부모 속성)
```css
display: flex;
justify-content: flex-start; 					/* 왼쪽으로 정렬[기본값] */
justify-content: flex-end; 						/* 오른쪽으로 정렬 */
justify-content: center;							/* 가운데 정렬 */
justify-content: space-between; 			/* 양 끝 정렬 후 균등분할 */
justify-content: space-around; 				/* 자식 주변으로 균등분할 */
justify-content: space-evenly; 				/* 균등분할 */

align-items: stretch;				/* [기본값] 자식중 높이가 가장 큰 자식에게 모든 자식을 맞춘다. */
align-items: center;				/* 가운데 맞춘다 */
align-items: flex-start;		/* 위에 맞춘다 */
align-items: flex-end;			/* 아래에 맞춘다 */

flex-direction: row | row-reverse | column | column-reverse
/* column 일 경우에는 justify-content, align-items가 바뀐다. */

flex-wrap: no-wrap | wrap | wrap-reverse
/* flex는 기본이 넘치지 않는다(자식의 width를 무시하면서 한줄에 넣는다) */
/* 자식요소가 넘치게(여러줄) 하려면 wrap 속성을 줘야 한다. */

align-content: stretch | flex-start | flex-end | center | space-between | space-around | space-evenly
/* 부모의 세로크기가 자식들보다 클때, 그리고 자식들이 여러줄로 표현될때 줄들을 어떻게 배치하는지에 대한 속성 */
```

## item(자식 속성)
```css
order: 0; 
/* 0 값이 작은 요소가 앞으로 배치된다. */
/* ... -2 -1 0 1 3 7 ... */

flex-glow: 0;
/* 0일때는 늘어나지 않는다. */
/* 1 이상의 값을 주면 값의 배수만큼 영역을 차지한다(늘어난다) */

flex-shrink: 1;
/* 기본값은 1 -> 1 이상일때는 줄어든다 */
/* 그래서 부모가 flex-wrap: nowrap이고 자식들이 부모크기를 넘쳐날때도 자식들이 줄어서 부모에서 한줄로 표현된다. */

flex-basis: auto;
/* width 대신에 사용할 수 있다. */

flex: flex-grow flex-shrink flex-basis;
flex: 0 1 auto; /* 기본값 */

align-self: inherit;
/* 부모 align-items에서 주어진 값을 나만 다른 값으로 표현하고자 할때... */
```
